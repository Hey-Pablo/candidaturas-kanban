# Candidaturas Kanban 📋

Kanban para organizar candidaturas a vagas de emprego. Feito com React + TypeScript + Vite.

## Funcionalidades

- **4 colunas Kanban**: 📌 Salvas, 📝 Inscrições, 🎯 Entrevista, ✅ Encerradas
- **Drag & drop** entre colunas com @hello-pangea/dnd
- Criar/editar/deletar cards com empresa, cargo, link, data, etiqueta e notas
- **Persistência automática** no localStorage
- **Tema escuro**
- Exportar dados como JSON
- **Responsivo** mobile-first

## Stack

- Vite 5 + React 18 + TypeScript
- Tailwind CSS 3 + shadcn/ui
- @hello-pangea/dnd
- lucide-react

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
